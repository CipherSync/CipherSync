About CipherSync

CipherSync is a powerful encryption/decryption package for Python and JavaScript. With its advanced algorithms and strong security features, CipherSync provides a secure and reliable solution for encrypting and decrypting data.

Python/pip is not yet supported. Please wait for the next release for the Python/pip version.

Installation

npm install ciphersync

Usage

To use CipherSync in your JavaScript project, you first need to require the module:

const cipherSync = require('ciphersync');

Then, you can use the cipherSync.encrypt() and cipherSync.decrypt() methods to encrypt and decrypt data:

// Encrypt a string with a key
const encryptedString = cipherSync.encrypt('my secret message', 'my secret key');

// Decrypt an encrypted message with a key
const decryptedString = cipherSync.decrypt(encryptedString, 'my secret key');
